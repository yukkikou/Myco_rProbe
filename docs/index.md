# Myco_rProbe

**Fungal rRNA Probe Design and Evaluation Pipeline**

Myco_rProbe is a comprehensive bioinformatics pipeline for designing, evaluating, and optimizing species-specific and pan-fungal rRNA-targeted hybridization probes. It supports single-species and multi-species probe design, circular RNA (circRNA) identification, and in silico probe evaluation using RNA-seq data.

---

## circRNA tools and pipelines

### [Myco_rProbe](about.md)
Design rRNA probes targeting conserved regions across fungal species with phylogenetic grouping, computational coverage evaluation, and taxonomic decision tree reports.
